## WHY

Forms are the primary way users input data into your application. If form controls (Inputs, Selects, Checkboxes) are poorly built, the user experience degrades instantly. 

A naive approach is to just style an `<input />` tag. However, a production-grade Design System must consider the surrounding ecosystem: Labels, Helper Text, Error Messages, Required Indicators, Leading/Trailing Icons, and Accessibility (ARIA roles). Building inputs that seamlessly integrate with form libraries (like React Hook Form or Formik) requires precise API design, specifically mastering `forwardRef`.

---

## THEORY

### The Field Context
An Input never exists in isolation. It is part of a "Field". A Field consists of:
1. **Label:** What is this input for? (Must be linked via `htmlFor` and `id`).
2. **Control:** The actual `<input>`, `<select>`, or `<textarea>`.
3. **Helper Text:** Additional context (e.g., "Must be at least 8 characters").
4. **Error Message:** Validation feedback (e.g., "Password is required").

Instead of forcing developers to piece these 4 elements together every time, a Design System should provide a holistic `<TextField>` component.

### Forwarding Refs
Modern form libraries (like React Hook Form) work by attaching a `ref` directly to the DOM `<input>` element to read its value uncontrollably (without forcing React state re-renders on every keystroke). If your custom Input component does not use `React.forwardRef` to pass the ref down to the underlying DOM node, these libraries will break.

### Accessibility (A11y)
When an input has an error, simply turning the border red is a failure of WCAG guidelines (color alone cannot convey state). 
- The input must have `aria-invalid="true"`.
- The error text must have an `id` (e.g., `id="email-error"`).
- The input must link to the error text using `aria-describedby="email-error"`.

## IMPLEMENTATION

Let's build a robust `TextField` component that handles labels, errors, icons, and `ref` forwarding flawlessly.

```tsx
import React, { forwardRef, useId } from 'react';
import styled, { css } from 'styled-components';

// --- STYLED COMPONENTS ---

const FieldWrapper = styled.div`
  display: flex;
  flex-direction: column;
  gap: 4px;
  width: 100%;
`;

const Label = styled.label`
  font-size: 14px;
  font-weight: 500;
  color: var(--text-primary);
  
  .required-asterisk {
    color: var(--color-danger-500);
    margin-left: 2px;
  }
`;

const InputContainer = styled.div<{ $hasError: boolean; $disabled: boolean }>`
  display: flex;
  align-items: center;
  border: 1px solid ${({ $hasError }) => ($hasError ? 'var(--color-danger-500)' : 'var(--border-subtle)')};
  border-radius: 6px;
  background: ${({ $disabled }) => ($disabled ? 'var(--bg-disabled)' : 'var(--bg-surface)')};
  transition: border-color 0.2s, box-shadow 0.2s;

  &:focus-within {
    border-color: ${({ $hasError }) => ($hasError ? 'var(--color-danger-500)' : 'var(--color-primary-500)')};
    box-shadow: 0 0 0 3px ${({ $hasError }) => ($hasError ? 'var(--color-danger-100)' : 'var(--color-primary-100)')};
  }
`;

const StyledInput = styled.input`
  flex: 1;
  border: none;
  background: transparent;
  padding: 10px 12px;
  font-size: 16px;
  color: var(--text-primary);
  outline: none; /* Focus is handled by the wrapper */

  &::placeholder {
    color: var(--text-placeholder);
  }
  
  &:disabled {
    cursor: not-allowed;
  }
`;

const IconSlot = styled.div`
  display: flex;
  align-items: center;
  padding: 0 12px;
  color: var(--text-secondary);
`;

const FeedbackText = styled.span<{ $isError?: boolean }>`
  font-size: 12px;
  color: ${({ $isError }) => ($isError ? 'var(--color-danger-500)' : 'var(--text-secondary)')};
`;

// --- COMPONENT API ---

export interface TextFieldProps extends React.InputHTMLAttributes<HTMLInputElement> {
  label: string;
  helperText?: string;
  error?: string;
  leftIcon?: React.ReactNode;
  rightIcon?: React.ReactNode;
}

export const TextField = forwardRef<HTMLInputElement, TextFieldProps>(({
  label,
  helperText,
  error,
  leftIcon,
  rightIcon,
  id: externalId,
  required,
  disabled,
  ...rest
}, ref) => {
  // Generate a unique ID if one isn't provided (Crucial for accessibility)
  const internalId = useId();
  const inputId = externalId || internalId;
  const helperId = `${inputId}-helper`;
  const errorId = `${inputId}-error`;

  const hasError = !!error;

  return (
    <FieldWrapper>
      <Label htmlFor={inputId}>
        {label} {required && <span className="required-asterisk" aria-hidden="true">*</span>}
      </Label>
      
      <InputContainer $hasError={hasError} $disabled={!!disabled}>
        {leftIcon && <IconSlot>{leftIcon}</IconSlot>}
        
        <StyledInput
          id={inputId}
          ref={ref}
          disabled={disabled}
          required={required}
          aria-invalid={hasError}
          aria-describedby={hasError ? errorId : helperText ? helperId : undefined}
          {...rest}
        />
        
        {rightIcon && <IconSlot>{rightIcon}</IconSlot>}
      </InputContainer>

      {/* Conditional Feedback Rendering */}
      {hasError ? (
        <FeedbackText id={errorId} $isError role="alert">
          {error}
        </FeedbackText>
      ) : helperText ? (
        <FeedbackText id={helperId}>
          {helperText}
        </FeedbackText>
      ) : null}
    </FieldWrapper>
  );
});

TextField.displayName = 'TextField';
```

## SYSTEM_DESIGN

### Integration with React Hook Form
Because our `TextField` correctly uses `forwardRef` and spreads the `...rest` props, it natively integrates with enterprise form state managers seamlessly.

```tsx
import { useForm } from 'react-hook-form';
import { TextField } from '@my-system/components';

function LoginForm() {
  const { register, handleSubmit, formState: { errors } } = useForm();

  return (
    <form onSubmit={handleSubmit(onSubmit)}>
      <TextField
        label="Email Address"
        type="email"
        // 'register' returns { onChange, onBlur, name, ref }
        // Because of forwardRef, the DOM node is correctly captured!
        {...register('email', { required: 'Email is required' })}
        error={errors.email?.message}
      />
    </form>
  );
}
```
