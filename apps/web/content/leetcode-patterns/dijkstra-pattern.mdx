## WHY
Shortest path with weights by BFS is wrong. Dijkstra greedily expands nearest via min-heap O((V+E)log V). Negative edges break it.

## THEORY
Pop min dist, relax neighbors, skip stale.
```mermaid
flowchart TD
 A[d=INF,src=0]-->B[pop min]
 B-->C[relax nbrs]
 C-->B
```
