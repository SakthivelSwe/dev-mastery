## WHY
Palindrome DP problems (Longest Palindromic Substring, Palindrome Partitioning) solve string questions by treating substrings s[i..j] as sub-problems. Expand around center or fill a 2-D DP table where dp[i][j] is true when s[i..j] is a palindrome.
