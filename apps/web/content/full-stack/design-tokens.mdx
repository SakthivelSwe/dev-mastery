### 1. Figma Workflow & Token Sync Pipeline
In a modern enterprise design system, Figma acts as the single source of truth for design tokens. Using **Figma Variables** or plugins like **Tokens Studio**, designers define primitive and semantic values.

```
[ Figma Variables ] 
       │
       ▼ (Export via API / Tokens Studio)
[ JSON Token Definitions ]
       │
       ▼ (Style Dictionary Build Step)
┌──────────────────────┼──────────────────────┐
│                      │                      │
▼                      ▼                      ▼
[ CSS custom props ]   [ Tailwind Config ]    [ TypeScript Types ]
```

#### GitHub Actions Automation
A GitHub Action automatically triggers on a Webhook from Figma, runs **Style Dictionary** to transform raw JSON files into multiple platform targets (CSS, JS/TS, iOS, Android), and opens a pull request:

```yaml
# .github/workflows/token-sync.yml
name: Sync Design Tokens
on:
  repository_dispatch:
    types: [figma-token-update]
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Setup Node.js
        uses: actions/setup-node@v3
        with:
          node-version: 18
      - run: npm ci
      - name: Build Tokens
        run: npm run build-tokens # Executes Style Dictionary
      - name: Create Pull Request
        uses: peter-evans/create-pull-request@v5
        with:
          commit-message: "chore: update design tokens from Figma"
          branch: tokens/figma-update
```

### 2. Token Definitions

#### CSS Custom Properties
```css
/* tokens.css */
:root {
  /* Primitives: Color */
  --color-blue-100: #dbeafe;
  --color-blue-500: #3b82f6;
  --color-blue-900: #1e3a8a;
  --color-gray-100: #f3f4f6;
  --color-gray-900: #111827;

  /* Semantics: Color */
  --color-bg-primary: var(--color-gray-100);
  --color-text-primary: var(--color-gray-900);
  --color-brand-primary: var(--color-blue-500);

  /* Primitives: Spacing */
  --space-1: 0.25rem; /* 4px */
  --space-2: 0.5rem;  /* 8px */
  --space-4: 1rem;    /* 16px */
  --space-8: 2rem;    /* 32px */

  /* Primitives: Typography */
  --font-family-sans: Inter, system-ui, sans-serif;
  --font-size-sm: 0.875rem;
  --font-size-base: 1rem;
  --font-size-lg: 1.125rem;
}
```

### 3. Accessibility Specifications (WCAG 2.1 AA)
- **Contrast Ratios**: Semantic color mappings must satisfy a minimum contrast ratio of **4.5:1** for normal text (under 18pt) and **3:1** for large text (18pt and above) or interactive UI components (borders, focus rings).
- **Spatial Focus Indicators**: Interactive elements must feature high-contrast outline tokens (`--color-brand-primary`) with at least a **2px** width and **2px** offset to ensure visible focus states.

### 4. React + TypeScript Implementation

```tsx
// TokenShowcase.tsx
import React from "react";

export interface TokenShowcaseProps {
  /** Optional container class overrides */
  className?: string;
}

export const TokenShowcase: React.FC<TokenShowcaseProps> = ({ className = "" }) => {
  const colors = [
    { name: "--color-brand-primary", value: "var(--color-brand-primary)", bgClass: "bg-brand" },
    { name: "--color-bg-primary", value: "var(--color-bg-primary)", bgClass: "bg-bg-primary" },
    { name: "--color-text-primary", value: "var(--color-text-primary)", bgClass: "bg-text-primary" },
  ];

  const spaces = [
    { name: "--space-1 (4px)", value: "var(--space-1)", tailwind: "h-space-1 w-space-1" },
    { name: "--space-2 (8px)", value: "var(--space-2)", tailwind: "h-space-2 w-space-2" },
    { name: "--space-4 (16px)", value: "var(--space-4)", tailwind: "h-space-4 w-space-4" },
    { name: "--space-8 (32px)", value: "var(--space-8)", tailwind: "h-space-8 w-space-8" },
  ];

  return (
    <div className={`p-6 space-y-8 max-w-4xl mx-auto ${className}`} data-testid="token-showcase">
      <section aria-labelledby="color-tokens-heading">
        <h2 id="color-tokens-heading" className="text-lg font-bold mb-4 text-text-primary">
          Semantic Color Tokens
        </h2>
        <div className="grid grid-cols-1 md:grid-cols-3 gap-4">
          {colors.map((color) => (
            <div key={color.name} className="border rounded-lg p-4 flex flex-col gap-2 shadow-sm bg-white">
              <div className={`h-12 w-full rounded border ${color.bgClass}`} />
              <span className="text-sm font-semibold">{color.name}</span>
              <span className="text-xs text-gray-500">{color.value}</span>
            </div>
          ))}
        </div>
      </section>

      <section aria-labelledby="spacing-tokens-heading">
        <h2 id="spacing-tokens-heading" className="text-lg font-bold mb-4 text-text-primary">
          Spacing Tokens
        </h2>
        <div className="space-y-4 bg-white p-4 border rounded-lg">
          {spaces.map((space) => (
            <div key={space.name} className="flex items-center gap-4">
              <div className="w-40 text-sm font-semibold">{space.name}</div>
              <div className="flex-1 bg-gray-100 p-1 rounded">
                <div className={`bg-brand rounded ${space.tailwind}`} />
              </div>
            </div>
          ))}
        </div>
      </section>
    </div>
  );
};
```
