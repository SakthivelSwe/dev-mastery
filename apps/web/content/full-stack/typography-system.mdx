### 1. Figma Workflow & Token Sync Pipeline
In Figma, typography is declared as structured Text Styles using a proportional typographic scale. Designers bind font size, line height, letter spacing, and font weights to responsive tokens.

```
Figma Typography Definition
  ├── Font Family: Inter
  ├── Scales: Minor Third (1.200) or Major Third (1.250)
  └── Fluid Range: 320px (Mobile) to 1440px (Desktop)
```

The design system architecture utilizes CSS `clamp()` functions to interpolate typography smoothly between viewport boundaries without relying on disjointed media query breakpoints.

### 2. Token Definitions

#### CSS Custom Properties
```css
/* typography.css */
:root {
  --font-sans: 'Inter', system-ui, -apple-system, sans-serif;

  /* Fluid typography definitions using clamp() */
  /* clamp(min, val, max) */
  --font-size-xs: clamp(0.75rem, 0.7rem + 0.25vw, 0.875rem);
  --font-size-sm: clamp(0.875rem, 0.82rem + 0.28vw, 1rem);
  --font-size-base: clamp(1rem, 0.95rem + 0.3vw, 1.125rem);
  --font-size-lg: clamp(1.25rem, 1.15rem + 0.5vw, 1.5rem);
  --font-size-xl: clamp(1.5rem, 1.35rem + 0.75vw, 2rem);
  --font-size-2xl: clamp(2rem, 1.8rem + 1vw, 3rem);

  /* Line heights */
  --line-height-none: 1;
  --line-height-tight: 1.25;
  --line-height-snug: 1.375;
  --line-height-normal: 1.5;
  --line-height-relaxed: 1.625;

  /* Font weights */
  --font-weight-normal: 400;
  --font-weight-medium: 500;
  --font-weight-semibold: 600;
  --font-weight-bold: 700;
}
```

### 3. Accessibility Specifications (WCAG 2.1 AA)
- **Minimum Font Size**: Body copy must not fall below **12px** (preferably **16px**) on any viewport.
- **Line Height Scaling**: Under WCAG Success Criterion 1.4.8, line height must be adjustable to at least **1.5 times** the font size, and paragraph spacing must be at least **2 times** the font size.
- **Reflow**: Viewports must support up to **400% zoom** without horizontal scrolling or text clipping.

### 4. React + TypeScript Implementation

```tsx
// Text.tsx
import React from "react";

export type TextVariant = "xs" | "sm" | "base" | "lg" | "xl" | "2xl";
export type TextWeight = "normal" | "medium" | "semibold" | "bold";
export type TextTag = "p" | "span" | "div" | "h1" | "h2" | "h3" | "h4" | "h5" | "h6";

export interface TextProps {
  as?: TextTag;
  variant?: TextVariant;
  weight?: TextWeight;
  className?: string;
  children: React.ReactNode;
}

export const Text: React.FC<TextProps> = ({
  as: Component = "p",
  variant = "base",
  weight = "normal",
  className = "",
  children,
}) => {
  const variantMap: Record<TextVariant, string> = {
    xs: "text-xs leading-tight",
    sm: "text-sm leading-snug",
    base: "text-base leading-normal",
    lg: "text-lg leading-snug",
    xl: "text-xl leading-tight font-bold",
    "2xl": "text-2xl leading-tight font-bold",
  };

  const weightMap: Record<TextWeight, string> = {
    normal: "font-normal",
    medium: "font-medium",
    semibold: "font-semibold",
    bold: "font-bold",
  };

  const combinedClasses = `${variantMap[variant]} ${weightMap[weight]} ${className}`.trim();

  return <Component className={combinedClasses}>{children}</Component>;
};
```
