# Dijkstra's Algorithm

Dijkstra's algorithm finds the shortest path from a single source vertex to all other vertices in a weighted graph with non-negative edge weights.

## 1. Core Mechanics & Theoretical Foundation

### The Greedy Property
Dijkstra's algorithm is greedy: at each step, it selects the unvisited vertex with the minimum tentative distance.
* **Proof of Correctness (Contradiction)**:
  Let $u$ be the next vertex chosen from the unvisited set, and let $d(u)$ be the tentative distance to $u$. Assume that the shortest path to $u$ is actually shorter than $d(u)$, passing through some unvisited vertex $y$.
  $$\text{Actual Path Weight} = d(y) + \text{weight}(y, \dots, u)$$
  Since all edge weights are non-negative, $\text{weight}(y, \dots, u) \ge 0$.
