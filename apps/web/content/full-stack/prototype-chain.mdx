# The Prototype Chain and Object Representation

JavaScript uses a prototype-based inheritance model. Every object maintains an implicit link to another object, forming a chain that is traversed during property lookups.

---

## 1. The `[[Prototype]]` Internal Slot and `__proto__`

Every JavaScript object has an internal slot called `[[Prototype]]`. This slot is a direct pointer to another object or `null`.
